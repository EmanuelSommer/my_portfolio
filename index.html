Hi I am Emanuel. I am a very curious and knowledge thirsty problem solver. Currently I do my M.Sc. in Mathematics in Data Science at the TUM that allows me to dive deep into two of the topics I am most passionate about: Machine Learning and Statistics. R Package containing tools for Data Quality testing + Shinydashboard

Following my internship at KPMG and with the gained experiance in Data Quality testing, I wanted to create my first shinydashboard and as the topic I naturally choose DQ testing. So I created an R package that contains the most important tools for the app. These tools include an implementation of the Local Outlier Factor Algorithm for multivariate outlier detection as well as many tools for univariate tests.

View Project

R Package for visualisation of P-values

The idea for this small project was born when many of the students of the course 'Statistics of Business Administration', where I was a tutor, had problems understanding the concept of P-values. Thus I created an R package that visualizes the P-values in the common setting of hypothesis tests for the 4 most common distributions. The usage, as it is meant for beginners in the field of statitics, is very straight forward.

Main Packages used: ggplot2, devtools
